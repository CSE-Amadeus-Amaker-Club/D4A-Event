📖 Control Commands

  • SET_SERVO_ANGLE (0x21): Set position for angular servos. Range: 0°-270°
  • SET_SERVO_SPEED (0x22): Set speed for rotational servos. Range: -100 (full reverse) to +100 (full forward)
  • STOP_SERVOS (0x23): Stop all rotational servos

Enable checkbox: include this channel in the command. Disabled channels are skipped (angle) or set to 0 (speed).

📖 Communication Protocol

All messages are echoed in web console as [WS TX] and [WS RX] in hex format. Commands are sent as binary packets with the following structure:
